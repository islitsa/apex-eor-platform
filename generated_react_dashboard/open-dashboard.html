Dashboard Debug Helper

Click the button below to open the dashboard in a new tab, then check the browser console for errors.

Instructions:

  1. Click the button above to open the dashboard
  2. Right-click on the blank page
  3. Select "Inspect" or "Inspect Element"
  4. Go to the "Console" tab
  5. Look for any red error messages
  6. Check if you see these debug messages:
    • "main.tsx is executing"
    • "Root element: [object HTMLDivElement]"
    • "Creating React root..."
    • "Rendering app..."
    • "App component is rendering"

Alternative: Check directly in iframe
