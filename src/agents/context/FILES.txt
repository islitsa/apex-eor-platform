WHAT EACH FILE DOES
====================

agent_studio.py
Purpose: Repository exploration infrastructure
Key classes: AgentStudio, SemanticIndex, SemanticArtifact
Main functions:
  - index_repository(): Scan and index repo
  - discover_data_sources(): Find datasets
  - discover_schemas(): Read actual schemas
  - explore_multi_hop(): Chain discovery
Size: 22KB, ~600 lines

context_swimming_agents.py  
Purpose: Agents that discover context autonomously
Key classes: ContextSwimmingAgent, ContextSwimmingUXDesigner, ContextSwimmingGradioImplementer
Main functions:
  - discover_context(): Autonomous exploration
  - design(): UX design with discovery
  - implement(): Code generation with discovery
Size: 19KB, ~450 lines

gradient_context.py
Purpose: Semantic navigation through gradient fields
Key classes: GradientContextSystem, GradientNavigator, GradientScore
Main functions:
  - compute_relevance_field(): Continuous scoring
  - compute_gradient_scores(): Direction + magnitude
  - navigate_gradient(): Follow gradient path
  - find_gradient_peaks(): Highest relevance
Size: 20KB, ~500 lines

integration_example.py
Purpose: Complete working demonstration
Key classes: AgentStudioOrchestrator
Main functions:
  - build_ui(): Full design → implementation
  - explore_semantic_space(): Gradient navigation demo
  - iterative_refinement(): Feedback loop demo
  - demo_traditional_vs_swimming(): Architecture comparison
Size: 11KB, ~300 lines
